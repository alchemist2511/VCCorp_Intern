## 📖 Giới thiệu

Repo này ghi lại quá trình học tập và thực tập của mình tại **VCCorp**, tập trung vào mảng **Data Engineering**: xây dựng pipeline thu thập dữ liệu, thiết kế mô hình dữ liệu, orchestration và đảm bảo chất lượng dữ liệu.

## 🗂️ Nội dung chính

- 📥 **Ingest** – Thu thập dữ liệu từ nhiều nguồn (API, crawl web, file) bằng Python
- 🏗️ **ETL/ELT Pipeline** – Xây dựng pipeline biến đổi dữ liệu với Spark
- ⏱️ **Orchestration** – Lên lịch và giám sát pipeline với Airflow
- ✅ **Data Quality** – Kiểm tra chất lượng dữ liệu 
- 📊 **Báo cáo** – Trực quan hóa dữ liệu với Power BI / Superset
